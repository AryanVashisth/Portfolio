# Aryan Vashisth - AI/ML Engineer Portfolio

Welcome to my professional engineering portfolio.

## Tech Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion & GSAP
- Spline 3D

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
